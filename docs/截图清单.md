## 拍摄顺序建议（30 分钟内跑通）

1. 启动 Lite 或 Docker → 浏览器登录/填入演示 API Key（若开启鉴权）  
2. 若库空：执行 seed 生成演示任务  
3. 按 01→08 顺序截图；再按需补 09–16  
4. 统一检查：无真实密钥、分辨率一致、中文 UI
