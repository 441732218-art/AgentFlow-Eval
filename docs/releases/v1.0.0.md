## 中文说明

### 主要功能

- **评测主链路**：任务 → 用例 → 异步/同步执行 → Trace → LLM-as-Judge / 规则评分 → 报告
- **Intelligence Center**：Dashboard 驾驶舱、Trace Explorer、故障诊断、Analytics、实时监控
- **企业能力**：多租户（请求头 `X-Tenant-ID`）、企业 RBAC、审计日志、API Key 鉴权
- **商业化骨架**：套餐 / 订阅 / 配额（HTTP 429）、Billing API、Stripe Checkout（mock，非生产收款）
- **Benchmark**：基准套件与跑分引擎（API + UI）
- **扩展**：插件系统（allowlist / 可选签名）、A/B 测试、多模态媒体评估
- **可观测**：Prometheus 指标路径 `/metrics`、AOLS 结构化日志、慢任务与 KPI
- **部署剖面**：`lite`（零 Redis Worker）/ `private`（Docker Compose）/ `saas`
- **交付形态**：Web/PWA、Electron 桌面（Windows / macOS）、Docker 镜像、离线交付包脚本

### 已知限制

1. **Stripe 为 mock 路径**：生产收款需自行配置真实 Stripe 密钥与 Webhook。
2. **桌面端不内嵌后端服务**：需配合 API 地址 / Lite 或 Docker。
3. **Windows 安装包未 Authenticode 签名**。
4. **插件动态加载**依赖 allowlist；签名校验为可选开关，默认严格模式视部署配置。
5. **测试覆盖率门禁约 40%**：持续提升中，尚未达到 80% 企业审计目标。
6. **macOS 产物当前以 arm64 为主**（Apple Silicon CI）。
7. **离线 Docker tar** 需本地执行 `scripts/export-offline-package.ps1` 生成，不随 git 发布。

- 请勿将含真实密钥的 `.env` / `.env.docker` 提交或打包进交付物。
- 生产请设置强 `SECRET_KEY`、`API_KEYS`、数据库密码，并开启 `AUTH_ENABLED=true`。
- 安全基线见 [SECURITY.md](https://github.com/441732218-art/AgentFlow-Eval/blob/main/SECURITY.md)、[docs/security-audit-report.md](https://github.com/441732218-art/AgentFlow-Eval/blob/main/docs/security-audit-report.md)。

## English Summary

**AgentFlow Intelligence v1.0.0** is an enterprise-oriented Agent evaluation platform with a full **task → trace → judge → report** pipeline, Intelligence Center UI, multi-tenant RBAC, billing skeleton, benchmarks, plugins, and deploy profiles: `lite` / `private` / `saas`.

### Download assets (this Release)

- Desktop: Windows NSIS / portable / zip; macOS arm64 dmg / zip (see SHA256 table above)
- Containers: `ghcr.io/441732218-art/agentflow-eval-backend:1.0.0` and `ghcr.io/441732218-art/agentflow-eval-frontend:1.0.0`
- Source: git tag `v1.0.0` + Compose private deploy (`DEPLOYMENT_GUIDE.md`)

### Known limits

- Stripe checkout is **mock** (not production payments)
- Desktop app is a **UI shell**; API must run separately (Lite or Docker)
- Windows builds are **not code-signed** (Authenticode)
- Coverage gate is about **40%** (not 80% yet)
