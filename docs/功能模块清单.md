## 3. 异步评测流水线（逻辑模块）

| 步骤 | 实现位置 | 说明 |
|------|----------|------|
| 入队 | `tasks` API → `TaskQueuePort` → Celery/Eager | 状态排队/运行；透传 `_trace_id` |
| 编排 | `evaluation/pipeline.py` + `celery_app/tasks.py` | 纯函数 + Worker |
| 逐条跑 Agent | `agent_runner`（factory → openai/http） | 生成 steps；AOLS `agent.*`/`llm.*`/`tool.*` |
| 落库 Trace | `celery_app/tasks.py` | 关联 test_suite |
| 自动 Judge | `judge_engine/llm_judge.py` + `metrics.py` | 规则 + LLM；可降级 |
| 计量 / 配额 | `billing` + `MeteringPort` | 可选 `BILLING_ENABLED` 门闩 |
| 汇总报告 | `reports` 聚合 | 前端报告页 |
| 可观测副作用 | metrics / slow_tasks / agent_logs | Dashboard & Monitoring |
